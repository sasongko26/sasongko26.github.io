Tipe data *char* di **rust** hanya dapat menampung 1 karakter nilai. Tidak sepeti tipe data lainnya, nilai *char* ditulis dengan diapit tanda kutip / petik satu.

```rust
fn main(){
   let inisial = 'S';
   println!("Alhamdullah... Korban {} telah sehat dan dapat menghadiri persidangan", inisial);
} 
```
