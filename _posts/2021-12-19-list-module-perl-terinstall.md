Untuk mengetahui semua modul **perl** yang terinstall bisa menggunakan **instmodsh**.
```shell
instmodsh
Available commands are:
   l            - List all installed modules
   m <module>   - Select a module
   q            - Quit the program
cmd?
```
Kemudian pilih l.
