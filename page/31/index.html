Mengetahui kapan komputer dinyalakan

25 Dec 2020
slackware / kernel / forensik

Untuk mengetahui kapan komputer/pc/laptop dinyalakan,gunakan dmesg

$ dmesg -T | head -1 
[Fri Dec 25 00:39:42 2020] Linux version 5.10.2

Penjelasan command :

  1. eksekusi sebagai root
  2. opsi -T pada dmesg bertujuan untuk memformat output dmesg dengan memberikan timestamp
  3. | adalah pipe, untu menjadikan output dmesg sebagai input untuk head
  4. head untuk menampilkan baris dari atas, jumlah baris yang ditampilkan sesuai angka yang dituliskan berikutnya
  5. [Fri Dec 25 00:39:42 2020] Linux version 5.10.2 # baris 1 yang kami tuliskan hanya sampai sini

Dari output tersebut tampak bahwa komputer dinyalakan pada hari Jum’at, 25 Desember 2020 sekitar jam 00.39.

Beralih ke Hugo

20 Dec 2020
slackware

Mengapa beralih dari jekyll ke hugo? Keunggulan hugo :

  1. Instalnya simpel. Hanya berupa binary. Jekyll butuh banyak dependensi
  2. tersedia di SBo lebih memudahkan dalam install, upgrade, remove kalau sudah tidak perlu
  3. cari pengalaman baru. selama 3 tahun ini menggunakan jekyll simpel,banyak dokumentasi dan theme yang free. Hugo setelah dicermati memberikan prospek yang tidak kalah bagus dibanding jekyll. Bagus di sini maksudnya adalah tergantung selera penulis. Hehehehe

Sedangkan kelemahan hugo adalah tidak didukung secara resmi oleh Github Page. Namun, tetap bisa menggunakan hosting github page. Begini proses migrasinya.

Akses WhatsApp Web via Seamonkey

22 Oct 2020
slackware / firefox

Beberapa waktu ini saya lebih sering browsing dengan Seamonkey karena lebih ringan dan cepat (penilaian subjektif). Overall lancar jaya aman terkendali. Namun, ada 1 yang tidak bisa dilakukan : akses WhatsApp Web!

Untuk mengatasinya,

  1. about:config
  2. Muncul peringatan. Klik I accept the risk!
  3. Isikan pada Search: general.useragent.compatMode.strict-firefox
  4. Ubah valuenya dari false ke true
  5. Restart Seamonkey
