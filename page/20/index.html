Aktivasi fancy indexing directory listing apache

Secara default, indexing pada apache hanya menampilkan nama direktori dan file yang berada dalam direktori tersebut dengan format unordered list. Bagi yang menganggap tampilan ini tidak menarik, bisa mengaktifkan fancy indexing dengan cara edit file /etc/httpd/httpd.conf, uncomment baris #Include /etc/httpd/extra/httpd-autoindex.conf kemudian restart apache # /etc/rc.d/rc.httpd restart
