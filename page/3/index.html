Tipe data string literal rust

Rust mengenal tipe data string literal (&str) atau string slice. Tipe data ini mirip dengan char, tetapi lebih banyak karakter yang bisa disimpan tidak hanya 1. Assignment dengan mengapit nilai variabel dengan tanda kutip. Contoh: fn main(){ let nama = "Sasongko"; println!("Halo, {}", nama); println!("Selamat belajar rust. Tetap semangat!"); }

Membuat tanda tangan digital

Tanda tangan digital sudah mulai banyak digunakan dan diterima masyarakat luas. Tanda tangan digital ini tidak hanya sekedar berupa foto atau scan tanda tangan, tetapi merupakan sertifikat orisinalitas. Mengapa sertifikat? Karena sebagai jaminan bahwa file yang diandatangani adalah asli dari yang bertanda tangan. Membuat tanda tangan digital di linux khususnya slackware sangat mudah. gpgsm --gen-key -o ettd.pem Command ini membuat sertifikat baru dengan nama ettd.pem.
