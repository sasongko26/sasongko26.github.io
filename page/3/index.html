Membuat tanda tangan digital

Tanda tangan digital sudah mulai banyak digunakan dan diterima masyarakat luas. Tanda tangan digital ini tidak hanya sekedar berupa foto atau scan tanda tangan, tetapi merupakan sertifikat orisinalitas. Mengapa sertifikat? Karena sebagai jaminan bahwa file yang diandatangani adalah asli dari yang bertanda tangan. Membuat tanda tangan digital di linux khususnya slackware sangat mudah. gpgsm --gen-key -o ettd.pem Command ini membuat sertifikat baru dengan nama ettd.pem. Tipe data bool rust

Tipe data bool di rust tidak berbeda dengan di bahasa pemrograman lainnya. Hanya ada 2 nilai, yaitu true atau false.
