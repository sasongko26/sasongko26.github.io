Disable direct cursor mode libreoffice writer

Direct cursor mode adalah suatu mode untuk menempatkan kursor di area mana saja pada halaman. Jika mode ini aktif dan kita melakukan klik di mana saja di suatu halaman, maka secara otomatis akan menempatkan kursor di tempat yang kita klik itu. Saya lebih suka mode ini non-aktif. Sehingga untuk mematikan mode ini klik menu Edit lalu hilangkan centang pada Direct Cursor Mode.
