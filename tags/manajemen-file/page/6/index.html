Melihat isi file zip

Menggunakan linux terutama slackware membuat pekerjaan lebih ringan dan efisien. Suatu ketika diberikan sebuah file terkompresi zip yang misterius. Misterius karena ukurannya besar dan konon kata empunya file tersebut ngehang saat akan dibuka di komputer dengan sistem bukan slackware juga bukan linux. Dan ketika dibuka di slackware ketahuanlah nama “penduduk” zip tersebut. Menggunakan zip zip -sf namafile.zip Atau dengan less less namafile.zip Atau bisa juga zipinfo namafile.zip
