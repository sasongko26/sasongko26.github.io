Operator perbandingan rust

Rust mengenal beberapa operator perbandingan, yaitu sama dengan (==), tidak sama dengan (!=), lebih kecil dari (<), lebih kecil atau sama dengan (<=), lebih besar dari (>), lebih besar atau sama dengan (>=). Hasil dari perbandingan ini adalah bool. Contoh: fn main(){ let a = 5; let b = 10; let c = a == b; let d = a != b; let e = a < b; let f = a <= b; let g = a > b; let h = a >= b; println!("a = b ? {c}"); println!("a != b ? {d}"); println!("a < b ? {e}"); println!("a <= b ? {f}"); println!("a > b ? {g}"); println!("a >= b ? {h}"); }

Deklarasi banyak variabel rust

Rust mengakomodir deklarasi beberapa variabel sekaligus dalam 1 statement. Caranya dengan memasukkan semua variabel ke dalam kurung dan antarvariabel diberi separator/pemisah tanda koma. Contoh: Deklarasi variabel satu persatu let negara = "Indonesia"; let tahun_merdeka: u16 = 1945; Kedua statement tersebut bisa diringkas let (negara, tahun_merdeka: u16) = ("Indonesia", 1945);

Operator aritmatika rust

Operator aritmatika yang dimiliki rust adalah penjumlahan (+), pengurangan (-), perkalian (*), pembagian, dan modulus atau sisa pembagian. Operasi aritmatika hanya bisa dijalankan terhadap variabel yang bertipe data sama. const A: f32 = 10.0; const B: f32 = 20.0; fn main(){ println!("A = {}", A); println!("B = {}", B); println!("A + B = {}", A+B); println!("A - B = {}", A-B); println!("A x B = {}", A*B); println!("A : B = {}", A/B); println!("A mod B = {}", A%B); println!("B mod A = {}", B%A); }

Konstanta rust

Konstanta dalam bahasa rust seperti variabel yang kita bisa mendeklarasikannya. Konstanta bisa mempunyai lingkup global; lebih luas dari pada variabel. Deklarasi konstanta harus dengan manifes yang menyebutkan eksplisit tipe datanya. Nama konstanta ditulis huruf kapital. Keyword-nya const atau static. const NEGARA: &str = "Indonesia"; fn main(){ const TANGGAL: u8 = 17; static BULAN: &str = "Agustus"; static TAHUN: u16 = 1945; println!("Negara {} merdeka pada {} {} {}", NEGARA, TANGGAL, BULAN, TAHUN); }
