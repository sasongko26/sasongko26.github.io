Reset pengaturan audio ke default

18 Dec 2024
slackware / multimedia

Secara default, apabila komputer dihidupkan (atau dihidupkan ulang), audio akan mengikuti pengaturan terakhir yang digunakan. Keunggulan dari sistem ini adalah pengguna tidak perlu repot mengatur kalau sudah menemukan konfigurasi yang cocok. Tapi hal ini buat sebgaian orang, terutama kami bisa menjadi masalah.

Soundcard yang saat ini digunakan adalah ES8336 yang problematik. Problem yang kami catat adalah tidak adanya suara yang terdengar, yang disebabkan oleh ketidakcocokan konfigurasi default alsa sehingga kami gunakan konfigurasi dari junocomp. Namun, konfigurasi dari junocomp ini ternyata juga masih ada masalah, yaitu selalu kembali ke pengaturan terakhir dalam pemilihan _audio device_nya. Contoh kasus: Kami pasang headset. Suara keluar dari headset. Laptop kami shutdown kemudian dihidupkan kembali dalam kondisi headset tidak lagi terpasang. Audio menjadi mati sempurna. Untuk mengatasi hal ini kami perlu mengembalikan ke pengaturan awal.

Mengakses file .dmg

04 Dec 2024
slackware / manajemen file

File .dmg biasanya adalah software installer yang berlaku untuk MacOs. Scara default pengguna linux tidak bisa mengakses isi dari .dmg ini. Namun, dengan software dan teknik tertentu hal ini bisa dilakukan.

Pertama, convert .dmg ke .img. Ini membutuhkan dmg2img. Khusus pengguna slackware dan derivatnya ini tersedia di SBo. Setelah dmg2img terinstall, konversi bisa dilakukan, dengan root privileges tentunya. Misalnya file .dmg nya adalah apel.dmg yang kemudian dikonversi menjadi apel.img.

$ dmg2img apel.dmg apel.img

Kemudian, atur apel.img sebagai loop device.

Disable direct cursor mode libreoffice writer

02 Dec 2024
slackware / office

Direct cursor mode adalah suatu mode untuk menempatkan kursor di area mana saja pada halaman. Jika mode ini aktif dan kita melakukan klik di mana saja di suatu halaman, maka secara otomatis akan menempatkan kursor di tempat yang kita klik itu. Saya lebih suka mode ini non-aktif. Sehingga untuk mematikan mode ini klik menu Edit lalu hilangkan centang pada Direct Cursor Mode.

Jalankan windows dari linux dengan qemu

28 Sep 2024
slackware / virtualisasi / qemu

Sebagian masyarakat memilih untuk dualboot atau bahkan multiboot dalam berkomputer. Satu Perangkat komputer diinstall lebih dari satu sistem operasi. Alhasil, harddisk pasti terbagi menjadi paling sedikit 2 partisi. Tak jarang, sistem operasi tersebut adalah linux dan windows. Kelemahan dari model seperti ini adalah pengguna harus memilih salah satu, mana sistem yang akan dijalankan. Namun, tidak demikian bila ada qemu.

Qemu memungkinkan sistem operasi windows dipanggil atau dijalankan dari linux. Untuk dapat menjalankannya perlu install qemu (tentu saja) dan edk2-ovmf. Keduanya tersedia di SBo.

Convert png ke pdf

13 Sep 2024
slackware / manajemen-file

Melengkapi catatan yang sudah ada tentang konversi file. Kali ini adalah konversi atau convert file dari berformat png menjadi pdf. Misalkan file gambar.png akan dikonversi menjadi pdf, maka command-nya

$ mogrify -format pdf gambar.png

Command tersebut di atas adalah bagian dari imagemagick yang secara default sudah terinstall di slackware.
