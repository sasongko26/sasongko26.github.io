Burn iso ke dvd

Misal, file slackware64-live-current.iso akan di-burn ke DVD. DVD terdeteksi oleh sistem sebagai /dev/sr0. Command dijalankan root. $ growisofs -dvd-compat -Z /dev/sr0=slackware64-live-current.iso
