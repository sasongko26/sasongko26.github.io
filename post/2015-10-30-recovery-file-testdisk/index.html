Mengembalikan File yang Terhapus dengan Testdisk

30 Oct 2015

Melanjutkan [catatan kemarin]({% post_url 2015-10-29-install-testdisk %}), kali ini adalah cara menggunakan testdisk untuk mengembalikan/memulihkan/recovery file yang terhapus. Misalkan dipunyai Flashdisk Toshiba berkapasitas 15 GB yang secara tak sengaja terhapus semua file yang di dalamnya.

Sebagai root jalankan

testdisk

kemudian akan muncul deskripsi tentang testdisk dan tawaran apakah akan membuat file log atau tidak. Di sini saya pilih No Log karena merasa log ini tidak terlalu penting.

Selanjutnya akan terdeteksi media simpan yang terpasang di komputer. Di sini flashdisknya terdeteksi sebagai /dev/sdc. pilih Disk /dev/sdc kemudian Proceed untuk melanjutkan.

Kemudian akan terdeteksi jenis partisinya. Di sini partisinya adalah Intel/PC Partition.

Kemudian muncul berbagai opsi, pilih Advanced

Selanjutnya, terlihat bahwa flashdisk ini hanya mempunyai 1 partisi dengan tipe FAT32 Pilih Undelete. Gunakan panah kanan kiri untuk berpindah opsi.

Akan muncul file dan foldernya. Gunakan panah kanan untuk membuka folder, q untuk keluar, : untuk menandai file yang dipilih, a untuk memilih semua file, C untuk menyalin semua file terpilih, c untuk menyalin file yang terpilih saat itu saja. Karena akan mengembalikan semua file, maka pilih a, kemudian C.

Akan muncul tawaran di mana file akan disalin ke mana.

Silahkan gunakan tanda panah untuk berpindah direktori, setelah ketemu atau mantap lokasinya tekan C untuk mulai menyalin.

Penyalinan sukses, tak ada 1 file pun yang gagal disalin/dipulihkan.

Setelah selesai, tekan q, kemudian Quit
