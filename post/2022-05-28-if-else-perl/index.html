If else perl

28 May 2022

Penyeleksian kondisi di perl dapat menggunakan syntax if…else dengan format

if ( kondisi ) {
	....
	} else {
	....
	}

Contoh

#!/usr/bin/perl

use strict;
use warnings;
use feature 'say';

my $username = "user";
if ( $username eq "nama" ) {
    say "Oke";
    } else {
    say "Ulangi lagi....";
    }
